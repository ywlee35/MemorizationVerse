# 📖 새일꾼반 성경암송 앱

사랑의교회 새일꾼반 성경 구절을 듣고 반복 재생하며 암송하기 위한 웹 기반 학습 앱입니다.

## ✨ 주요 기능

- 📝 **11주차 커리큘럼**: 매주 2개씩 총 22개 성경 구절
- 🎵 **제목 + 구절 순서 재생**: 매 재생 시 주제 제목 음성 후 구절 재생
- 🔂 **개별/전체 재생 선택**: 1절만 · 2절만 · 전체(1절+2절) 선택 가능
- ♾️ **무한반복 포함 반복 옵션**: 1·2·3·5·7·10·20회, 무한반복(∞) 지원 (기본값: ∞)
- ⚡ **재생 속도 조절**: 1x, 1.25x, 1.5x, 2x 지원
- ⏩ **구간 탐색**: 진행도 바 클릭으로 원하는 구간으로 이동
- 🎨 **다크 테마**: 장시간 사용에 편한 프리미엄 다크 모드 디자인
- 📱 **모바일 최적화**: 모든 기기에서 완벽하게 작동

## 📂 파일 구조

```
index.html              # 메인 앱 파일
Audio/
  01_1.mp3              # 1주차 제목 음성
  01_2.mp3              # 1주차 1절 음성
  01_3.mp3              # 1주차 2절 음성
  02_1.mp3 ~ 11_3.mp3  # 2주차 ~ 11주차 (동일 패턴)
README.md               # 이 파일
```

> **오디오 파일 명명 규칙**: `주차번호_트랙번호.mp3`
> - `_1` = 주제 제목, `_2` = 1절(첫 번째 구절), `_3` = 2절(두 번째 구절)

## 🚀 사용 방법

### 1. **주차 선택**
- 상단 탭에서 원하는 주차를 클릭합니다
- 해당 주차의 주제와 성경 구절 2개가 표시됩니다

### 2. **재생 모드 선택**
- **1절만**: 제목 → 첫 번째 구절 재생
- **2절만**: 제목 → 두 번째 구절 재생
- **전체**: 제목 → 1절 → 2절 순서로 재생

### 3. **재생**
- ▶ 버튼을 눌러 재생합니다
- 각 구절 카드의 ▶ 버튼으로 해당 절만 바로 재생할 수 있습니다
- 재생 중인 구절 카드는 금색으로 하이라이트됩니다

### 4. **반복 횟수 설정**
- 반복 버튼을 클릭하여 반복 횟수를 선택합니다
- 기본값은 **무한반복(∞)**

### 5. **재생 속도 조절**
- `1x` 버튼을 클릭하여 속도를 변경합니다
- 처음 암송 시 1x, 익숙해지면 1.25x~1.5x 추천

### 6. **주차 이동**
- ⏮ / ⏭ 버튼으로 이전/다음 주차로 이동
