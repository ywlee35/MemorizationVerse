# 📖 성경 암송 앱

성경 구절을 반복 재생하며 암송하기 위한 웹 기반 학습 앱입니다.

## ✨ 주요 기능

- 📝 **11주차 커리큘럼**: 매주 2개씩 총 22개 성경 구절
- 🎵 **음성 반복 재생**: 자동 반복 기능 (1회, 2회, 3회, 5회, 7회, 10회)
- ⚡ **재생 속도 조절**: 1x, 1.25x, 1.5x, 2x 지원
- ⏩ **구간 탐색**: 진행도 바 클릭으로 원하는 구간으로 이동
- 🎨 **어두운 테마**: 장시간 사용에 편한 다크 모드 디자인
- 📱 **모바일 최적화**: 모든 기기에서 완벽하게 작동

## 📂 파일 구조

```
성경암송_앱.html          # 메인 앱 파일 (브라우저에서 열기)
01.mp3 ~ 11.mp3         # 성경 구절 음성 파일 (11주차)
README.md               # 이 파일
```

## 🚀 사용 방법

### 1. **앱 실행**
- `성경암송_앱.html` 파일을 웹 브라우저에서 엽니다
- (Chrome, Firefox, Safari, Edge 등 모든 모던 브라우저 지원)

### 2. **주차 선택**
- 상단 탭에서 원하는 주차를 클릭합니다
- 해당 주차의 성경 구절 2개가 표시됩니다

### 3. **재생**
- ▶ 버튼을 눌러 음성을 재생합니다
- 자동으로 설정된 반복 횟수만큼 반복됩니다

### 4. **반복 횟수 설정**
- 🔁 버튼을 클릭하여 반복 횟수를 선택합니다
- (기본값: 3회)

### 5. **재생 속도 조절**
- `1x` 버튼을 클릭하여 속도를 변경합니다
- 선택지: 1x, 1.25x, 1.5x, 2x
- 청력가가 아닌 경우 느린 속도(1x, 1.25x) 추천

### 6. **주차 이동**
- ⏮ / ⏭ 버튼으로 이전/다음 주차로 이동

## 💡 학습 팁

### 축음기(Spaced Repetition) 학습법 활용
1. **1단계**: 첫 번째 주차부터 시작 (3회 반복)
2. **2단계**: 한 주 후 이전 주차 복습 (2회 반복)
3. **3단계**: 두 주 후 더 이전 주차 복습 (1회 반복)

### 추천 학습 일정
- **월~금**: 새로운 주차 음성 3회 반복 재생
- **토**: 지난 주 복습
- **일**: 전주 복습
